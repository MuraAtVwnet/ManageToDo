作業中に割り込みでタスクを依頼された際に、手早く割り込み依頼を todo リストに登録して、作業が一段落したときに todo 内容を確認して割り込みタスクの対応をしたいです
この todo ツールを PowerShell で作ってください
仕様は以下でお願いします

・手早く割り込みタスクのタイトルと内容を登録できる
・登録したタスクを一覧で確認できる
・完了したタスクを一覧にしないようにする
・間違って完了にしたタスクを簡単に未完了に戻すことができる
・タスクリストを保存し、ツールを起動したときに保存したタスクリストが読み込まれる
・タスクの保存は、タスクの追加更新など更新操作の都度保存してデータロストが起きないようにする
・GUI で操作するようにする
・将来的に手動でモジュールとして登録するので、現段階は global 関数の .ps1 として実装してください
・関数は1つにして、関数名を ToDo としてください
・タスクの完了操作はチェックボックスで出来るようにしてください
・内容の全文を確認できるようにしてください。ただし一覧表示は先頭一部の表示で構いません
・「完了タスクも表示」はトグルボタンにしてください
・完了タスクを物理削除する機能を追加してください
・完了タスクを物理削除は、未完了タスク表示 ON の時だけ使用できるようにして
・他の Windows に隠れないようににして
・内容に URL が含まれている場合は、URL をリンクにして、クリックすると Web ブラウザで Web Page が開けるようににして
・登録したタスクを修正できるようにしてください
・タスクを登録する際に、期限を任意で入力できるようにして下さい
・期限は年月日の様式で、年月を省略して日だけ入力した場合はその日付から最も近い将来の日付にし、年を省略して月日だけ入力した場合はその月日から最も近い将来の日付にしてください
・期限が入力されたタスクは、ほかのタスクよりリスト表示で上位に表示されるようにしてください
・リスト表示の並び順は、期限が小さい +登録日が大きい順に並ぶようにして完全コードを出力してください
・タスクの入力/編集を中止するため入力クリアボタンを追加し、入力クリアボタンを押すと入力内容がクリアされるようにしてください
